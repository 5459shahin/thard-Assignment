What is Difference between Flexbox and grid?

Grid is a two-dimensional layout system with rows and columns. It is also used for the responsive web page. Grid-template-column is used for responsiveness for the device screen.Flex is a one-dimensional layout. It works with all kinds of display devices and screen sizes. Flexbox mostly helps align content & move blocks.

What are the difference between Bootstrap & Tailwind?

Tailwind-A utility-first CSS framework packed with classes like flex, pt-4, text-center and rotate-90 that can be composed to build any design, directly in your markup.
Bootstrap-A Component Based CSS framework opensource and totally free.

What is css Box-model?

The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.

What is sementic Tag?

Which elements have meaningful details are called semantic elements. Semantic elements are meaningful elements. These tags provided meaning. Some semantic tags are nav, header, footer, aside, section, etc.
